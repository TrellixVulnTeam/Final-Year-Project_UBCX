API Query Page

This is the second stage of my project. Currently, this API acquires the population data from the World Bank's database.

Upon using the API to search for your data, the data will be represented in a JSON format.

In the future, I hope to allow the user to query from multiple sources as well as having the ability to download the data in multiple formats.

Instructions to search for population data are as follows:

You can search for the population of any country with the following adjustable parameters:

  • Country
  • Year
  • Gender
  • Age Range

.../populationURL/?country=__(1)__&year=__(2)__&indicator=__(3)__&age=__(4)__

  1. You must select a country code from the list.
  2. Enter a year between 1960 - 2016
  3. You can pick a gender indicator: Male / Female / Total
  4. Select from an age range: 0-14 / 15-64 / 65-

You must select the "Country" and the "Year" parameters for this API to function.

You may then choose between including the "Indicator" parameter on its own, or the "Indicator" and "Age" parameters together.

*The API search cannot include "Age" without including the "Indicator" parameter also*


For example:

The query to find the population of Females aged between 15-64, in Ireland in 1996 would be:

".../populationURL/?country=IRL&year=1996&indicator=Female&age=15-64"

The results will print in JSON format. The result of the example above will be: 

{
    "age": "0-14",
    "country": "Ireland",
    "indicator": ".MA.IN",
    "population": "493147",
    "year": "2011"
}
